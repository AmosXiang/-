# P1b 布局重构验收证据（CC · 2026-07-15）

方案依据：集成计划批次表 P1b + v2.1 §三。基线：`feature/camera-derive@24e85f5`（批次 1 集成后主线）。
验证：`tsc --noEmit` 通过 + dev 真机 DOM 检查，控制台（error 级）零输出。

## 交付内容与验证结果

### 1. 分析页布局重构
- **左栏 = 分镜时间轴**：当前视频全部分镜（时间码 + 运镜 + 描述），点击联动主区选中分镜；
  原左栏的上传区/筛选/记录列表/剧本列表全部退出分析页（已归素材库首页）。
  实测：`hasTimeline=true`、旧"本地视频分析库"侧栏不再渲染。
- **右栏统计默认收起**：默认不渲染右栏、主区占满（`workspace-span`）；顶部 tabs 栏显示一行摘要
  `N 分镜 · 平均 xs · N 角色` + 「展开分析摘要」开关，点击后右栏恢复。
  实测：默认 `inspectorHidden=true` → 点击展开后统计面板出现。
- **导出 JSON 迁移**：顶栏"导出 JSON 报告"按钮移除，改为分析页 tabs 栏内的小按钮（系统抽屉不放业务导出，遵守 v2.1 抽屉边界）。

### 2. 系统状态抽屉
- 原 ComfyUI 锚定弹层升级为**右侧固定抽屉**（320px、带遮罩、标题"系统状态"、关闭按钮）；
- 新增**生成任务队列摘要**：排队 / 生成中 / 失败 三格计数（数据源 comfyTasks）。
  实测：`position:fixed; right:0; width:320px`，标题与队列区渲染正常。

### 3. tab 进 URL
- 路由扩展 `#/analysis/:id?tab=shots|characters|narrative`；tab 点击写 URL，刷新/直达按 URL 恢复。
  实测：直达 `?tab=characters` → "人物画像" tab 激活。

### 4. 项目卡"继续上次工作"
- `#/studio` 列表卡新增进度摘要：分镜定稿 x/N（全定稿变绿）、角色画像 x/N、"N 镜基于旧输入"（isStale，有则显示琥珀色）、"继续编辑 →"。数据来自 WP-B 合入的 Shot 字段，纯前端派生。
- 创意工作室左栏改为**项目切换器**（含定稿进度行），上传区退出。
  实测：`progressChips/resumeBtn=true`，3 张项目卡渲染正常。
